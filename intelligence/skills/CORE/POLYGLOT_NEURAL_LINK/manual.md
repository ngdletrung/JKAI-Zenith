## 🏛️ 1. PHẠM VI CHIẾN THUẬT
- **Java/Kotlin**: Hệ sinh thái Enterprise, JVM, Spring.

## 📐 2. NGUYÊN TẮC THỰC THI
- **Code Clean**: Luôn viết code dễ đọc, có comment đầy đủ.
- **Performance First**: Luôn cân nhắc độ phức tạp thuật toán (Big O).
- **Security Mindset**: Luôn kiểm tra các lỗi bảo mật tiềm ẩn như Buffer Overflow hay SQL Injection.
