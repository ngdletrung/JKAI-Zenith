# 🌐 JKAI ZENITH: MANUAL - THẤU THỊ KIẾN TRÚC (`skill_deep_context`)

## 🎯 MỤC TIÊU
Phân tích toàn diện sơ đồ hạ tầng Docker, các kết nối mạng (Network), các phân vùng lưu trữ (Volumes) và mối quan hệ phụ thuộc giữa các Service trong hệ sinh thái JKAI.

## 🛠️ CÁCH SỬ DỤNG
Master có thể gọi kỹ năng này để:
1. Kiểm tra cấu trúc `docker-compose.yml`.
2. Truy vết các kết nối mạng giữa các Container.
3. Phát hiện các điểm nghẽn hoặc cấu hình sai trong hạ tầng.

## 📡 THAM SỐ (ARGS)
- `focus`: (Optional) Tên service cụ thể cần phân tích sâu.
- `detail_level`: "high", "medium", "low".

## 💎 KẾT QUẢ
Trả về bản đồ quan hệ (Relationship Map) và các khuyến nghị tối ưu hóa kiến trúc hạ tầng. 💎🫡🦾🚀🌌
