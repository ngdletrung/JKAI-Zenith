# 🌀 JKAI ZENITH: WORKFLOW - THẤU THỊ KIẾN TRÚC (`skill_deep_context`)

## 🛡️ QUY TRÌNH THỰC THI (STEP-BY-STEP)

1. **Giai đoạn 1: Thu thập Dữ liệu (Scanning)**
   - Quét file `docker-compose.yml` tại root directory.
   - Truy vấn thông tin thực tế từ `docker network ls` và `docker inspect`.

2. **Giai đoạn 2: Phân tích Quan hệ (Dependency Mapping)**
   - Xây dựng đồ thị phụ thuộc (Depends_on).
   - Xác định các cổng (Ports) đang mở và giao thức kết nối.

3. **Giai đoạn 3: Thẩm định Cấu trúc (Architectural Audit)**
   - Kiểm tra các biến môi trường (Env) quan trọng.
   - Kiểm tra phân vùng lưu trữ (Volumes) để đảm bảo tính toàn vẹn dữ liệu.

4. **Giai đoạn 4: Tổng hợp & Đề xuất (Synthesis)**
   - Xuất báo cáo sơ đồ hạ tầng chuẩn Elite.
   - Đề xuất các phương án tối ưu hóa Network hoặc tài nguyên.
