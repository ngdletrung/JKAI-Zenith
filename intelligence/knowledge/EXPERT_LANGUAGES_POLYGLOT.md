# 🌍 EXPERT LANGUAGES POLYGLOT: THE ELITE CODEX 📚

Bộ quy tắc tiêu chuẩn để JKAI Zenith đọc, viết và đánh giá mã nguồn đa ngôn ngữ theo phong cách Senior Engineer.

## 🐍 1. PYTHON: THE INTELLIGENCE CORE
- **Zenith Style**: Sử dụng Type Hints, Docstrings chuẩn Google và ưu tiên Asyncio cho các tác vụ I/O.
- **Optimization**: Tận dụng List Comprehensions, Generators và tránh lặp lại code (DRY).
- **Security**: Tuyệt đối không hardcode bí mật (Secrets), sử dụng `.env`.

## ☕ 2. JAVA & KOTLIN: ENTERPRISE ROBUSTNESS
- **Structure**: Tuân thủ triệt để các Design Patterns (Singleton, Factory, Observer).
- **Safety**: Xử lý ngoại lệ (Exception Handling) đa tầng, tránh NullPointer bằng cách sử dụng `Optional`.
- **Build**: Thông thạo Maven/Gradle và cơ chế Dependency Injection.

## ⚙️ 3. C++ & RUST: PEAK PERFORMANCE
- **Memory**: Quản lý vùng nhớ thủ công (C++) hoặc thông qua cơ chế Ownership/Borrowing (Rust).
- **Concurrency**: Xử lý đa luồng (Multi-threading) an toàn, tránh Race Conditions.
- **Modern C++**: Sử dụng Smart Pointers (unique_ptr, shared_ptr) và chuẩn C++17/20.

## 🌐 4. JS/TS: DYNAMIC UNIVERSE
- **Modern JS**: Ưu tiên ES6+, Arrow Functions và Destructuring.
- **TypeScript**: Ép kiểu chặt chẽ (Strict Mode), sử dụng Interfaces và Enums để quản lý trạng thái.
- **React Standards**: Hooks (useEffect, useMemo), Component Reusability và State Optimization.
