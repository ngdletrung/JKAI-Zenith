# JKAI Mission Control – Frontend (Liquid Glass)

Hệ thống giao diện quản trị tối cao của JKAI, được xây dựng trên nền tảng React + Vite với ngôn ngữ thiết kế **Liquid Glass**.

## 🎨 Design System
- **Framework**: React 18 + TypeScript
- **Styling**: Tailwind CSS + Framer Motion (Animations)
- **Visuals**: Neural Pipeline (ReactFlow), Trinity Grid layout.
- **Theme**: Deep Space (Nebula effects).

## 🚀 Phát triển
1. **Cài đặt**: `npm install`
2. **Chạy dev**: `npm run dev`
3. **Build**: `npm run build`

## 🧩 Cấu trúc
- `/src/components`: Chứa các thành phần UI (DAGViewer, CustomNode...).
- `/src/hooks`: Logic WebSocket và xử lý Task.
- `/src/index.css`: Định nghĩa hệ thống Glassmorphism và hiệu ứng Nebula.
